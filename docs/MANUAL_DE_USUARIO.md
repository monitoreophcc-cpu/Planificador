# 🧭 Guía Rápida — Nexo — Control Operativo

> **(3 minutos para empezar, el resto es automático)**
>
> Esta guía está pensada para usar el sistema sin aprenderlo.
> Léela una vez. Luego solo trabaja.

---

## 🧩 ¿Qué es cada pantalla?

### 📝 Registro Diario
**Lo que está pasando HOY.**
Aquí registras tardanzas, ausencias, errores y cualquier novedad real del día.
👉 *Es la pantalla que usarás casi todo el tiempo.*

### 📅 Planificación
**Lo que debería pasar.**
Sirve para organizar semanas futuras, vacaciones y revisar cobertura.
👉 *Aquí no se registran hechos reales, solo planificación.*

### 📊 Reportes
**Lo que ya pasó.**
Resumen mensual, puntos, historial y respaldo para RRHH o gerencia.
👉 *No modifica nada. Solo muestra la verdad registrada.*

### ⚙️ Configuración
**Las reglas del sistema.**
Personas, feriados, cobertura mínima y opciones avanzadas.
👉 *No es para el día a día.*

---

## 📝 Registrar incidencias (uso diario)

Cuando alguien llega tarde, falta o tiene una novedad:
1.  Busca a la persona en la lista.
2.  Selecciona el tipo de incidencia.
3.  (Opcional) Escribe una nota breve.
4.  Presiona **Registrar**.

### ✔️ Qué hace el sistema automáticamente
*   Guarda la fecha y hora exacta.
*   Calcula impacto en cobertura.
*   Aplica reglas (vacaciones, licencias, feriados).
*   Muestra alertas si algo queda descubierto.

### ↩️ ¿Te equivocaste?
Después de registrar, aparece **Deshacer** por unos segundos.
Úsalo. Es la forma correcta de corregir errores recientes.

---

## 📅 Planificación semanal

Usa esta vista para preparar semanas futuras o revisar cargas.

### Cómo leer la grilla
*   **Celdas normales:** planificación estándar.
*   **Celdas con color:** incidencias ya registradas.
*   **Indicadores rojos:** falta personal ese día.

> 👉 **Si ves rojo, el sistema no te castiga, te avisa.**

### Cambios comunes
*   Usa las acciones disponibles (menú) para gestionar swaps o coberturas.
*   **Vacaciones:** el sistema cuenta solo días laborales automáticamente.
*   No necesitas revisar feriados externos.

---

## 📊 Reportes y cierre de mes

Cuando necesites responder preguntas como: *"¿Quién faltó este mes?"* o *"¿Cuántos puntos tiene cada persona?"*.
Ve a **Reportes**.

### Qué encontrarás
*   Resumen mensual por persona.
*   Historial completo.
*   Datos listos para copiar o exportar.
*   Registro verificable de lo ocurrido.

> 👉 **Esto es tu respaldo. No depende de memoria ni Excel.**

---

## ⚠️ Edición avanzada (usar con cuidado)

Algunas acciones especiales requieren **Edición Avanzada** (en Configuración o Reportes).

### Qué significa
*   Permite corregir registros pasados.
*   **No borra la historia** (siempre queda rastro).
*   Toda acción queda registrada.

> 👉 **Esto existe para correcciones excepcionales, no para uso diario.**
> Si no sabes si debes usarlo, probablemente no lo necesitas.

---

## 🧠 Cómo pensar el sistema (importante)

*   El sistema no adivina.
*   El sistema no corrige personas.
*   **El sistema registra la realidad.**

> **Si algo se ve mal, es una señal operativa, no un error del sistema.**

### ✅ Reglas simples para trabajar tranquilo
1.  Usa **Registro Diario** para lo que pasó.
2.  Usa **Planificación** para lo que vendrá.
3.  Usa **Deshacer** cuando te equivoques.
4.  Confía en las alertas.
5.  No fuerces el sistema: ya está haciendo el trabajo pesado por ti.

---

> **Consejo final**
> Si todo está en verde, sigue trabajando.
> Si algo se pone rojo, atiéndelo.
> **El sistema no estorba: te cuida la espalda.**
