# **App Name**: Call Analytics Dashboard

## Core Features:

- CSV/Excel File Upload: Allows users to upload call data from CSV and Excel files.
- KPI Calculation: Calculates key performance indicators (KPIs) from the uploaded data, globally and by shift.
- Data Table Rendering: Renders data in tabular format.
- Chart.js Integration: Generates charts and graphs using Chart.js to visualize call data.
- Global State Management: Manages application state using Zustand.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) to convey professionalism and trust, in line with data analysis. 
- Background color: Light gray (#F0F4F8), offering a clean, neutral backdrop for the data-heavy interface.
- Accent color: Soft orange (#DCA170) to highlight interactive elements.
- Body and headline font: 'Inter', a sans-serif, chosen for its modern, neutral appearance and suitability for both headlines and body text in data-rich dashboards.
- Use a consistent set of minimalistic icons.
- Responsive layout to ensure compatibility across different devices.
- Subtle transitions and animations to improve user experience.
